Operators
=========

Operators who wish to run Casper Network infrastructure can explore the following sections.

We recommend that you have prior knowledge of Unix-based operating systems and proficiency with *systemd* and *bash* scripting. If you are unfamiliar with *systemd*, the `Arch Linux page on systemd <https://wiki.archlinux.org/index.php/systemd>`_ is a good introduction.
